This trait is mainly used to help setup debian live for the paella installer, however, the configuration should be flexible enough to be used to help setup live helper to create a specific type of live system.  The configuration that is specific for the paella installer will be in another trait that depends on this trait.  If you wish to use live-helper to make many different types of live systems, then this trait is probably only useful for the packages that it contains, and for setting up common defaults for those systems.
